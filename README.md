# Python-Scientific-Calculator
A feature-rich scientific calculator built with Python and Tkinter, supporting arithmetic operations, scientific functions, keyboard shortcuts, and a modern dark theme UI.

## Features

✅ Basic Arithmetic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

✅ Scientific Functions
- Square Root (√)
- Square (x²)
- Logarithm (Log)
- Factorial (!)
- Sin
- Cos
- Tan
- Pi (π)
- Euler's Number (e)

✅ Other Features
- Percentage (%)
- Clear (AC)
- Backspace (⌫)
- Decimal Point (.)
- Keyboard Support
- Dark Theme UI

## Technologies Used

- Python
- Tkinter
- Math Module

## How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Scientific-Calculator.git
```

2. Open the project folder

```bash
cd Scientific-Calculator
```

3. Run the file

```bash
python calculator.py
```
